What is CityTropes?


The website is named after my GitHub username. You can find my repositories on github.com/CityTropes.


This is my first HTML5/CSS3 website. It's not much, but I tried to implement some "best practices", like responsive design, flexbox, media queries and semantic elements. I used the following sources to learn the basics of HTML5/CSS3 and to build this website:

  • internetingishard.com
  • w3schools.com/howto


The CityTropes website is part of a prep course to prepare for Switchfully's Java developer traineeship: "Create a small website in which you introduce yourself & present a topic you are interested in".
